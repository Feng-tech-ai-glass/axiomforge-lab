# AxiomForge Lab - Open Source Layer

## What is in this repo (SAFE to share)
- Demo frontends (pattern inventor, puzzle inventor)
- Evaluation tools
- Documentation
- Simplified examples

## What is NOT in this repo (NEVER upload)
- Core DNA implementation
- Model weights (.pt files)
- Experiment data (.jsonl)
- API keys
- Passwords or credentials
- Memory data
- Self-evolution scripts

## Pre-upload checklist
- [ ] No API keys in any file
- [ ] No .pt weight files
- [ ] No .jsonl experiment data
- [ ] No passwords or credentials
- [ ] No core strategy implementations
